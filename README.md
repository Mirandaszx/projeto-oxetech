# Diário de Treino Iron Pump

Projeto de front-end desenvolvido com HTML, CSS, Bootstrap e JavaScript puro para organizar uma rotina semanal de treinos.

## Objetivo

Criar um diário de treino simples e responsivo, permitindo cadastrar exercícios, acompanhar o progresso e registrar o histórico de execução em uma interface prática. Como se trata de uma atividade de front-end, os dados existem apenas durante a sessão aberta da página.

## Funcionalidades

- Adicionar exercícios dinamicamente.
- Editar e remover exercícios cadastrados.
- Marcar exercícios como concluídos.
- Filtrar por grupo muscular.
- Buscar exercícios pelo nome.
- Aplicar modelos rápidos de treino `Push`, `Pull` e `Legs`.
- Visualizar a divisão semanal do treino.
- Acompanhar o progresso semanal por porcentagem.
- Registrar o histórico de exercícios concluídos.
- Usar cronômetro de execução do treino.
- Validar os campos obrigatórios antes do cadastro.
- Exibir um resumo com total de exercícios, grupos musculares trabalhados e volume estimado.
- Funcionar bem em desktop e celular.

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Preencha o formulário com os dados do exercício.
3. Clique em `Adicionar exercício` para salvar.
4. Use os filtros, o cronômetro e os modelos rápidos conforme necessário.

## Tecnologias utilizadas

- HTML semântico
- CSS com Flexbox e Bootstrap
- JavaScript puro para manipulação do DOM
